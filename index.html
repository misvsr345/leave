📤 Submit
ଆପଣ ଛୁଟି ଆବେଦନ କରିଲେ, ଏହା MIS Executive ଙ୍କ ପାଖକୁ ଯିବ। ଆପଣଙ୍କ screen ରେ ତୁରନ୍ତ result ଦେଖାଯିବ।
✅ Approved
ଯଦି ଆପଣଙ୍କ ଛୁଟି Approved ହୋଇଛି, ଆପଣ ଛୁଟି ନେଇ ପାରିବେ। Leave ID ସାଇତି ରଖନ୍ତୁ।
💰 Leave Balance
ପ୍ରତି ମାସ ଜଣ ପ୍ରତ୍ୟେକ employee ଙ୍କୁ 1.5 ଦିନ Casual Leave ମିଳିବ। ଅଧିକ ଛୁଟି ଦରକାର ହେଲେ Sick Leave / SPL ବ୍ୟବହାର କରନ୍ତୁ।
❌ Rejected / Cancel
ଯଦି ଛୁଟି Reject ହୁଏ, screen ରେ କାରଣ ଦିଆଯିବ। MIS Executive ବା HR ଙ୍କ ସହ ଯୋଗାଯୋଗ କରନ୍ତୁ।
⏳ Pending / Absent
Absent mark ହେଲେ balance ରୁ penalty কাटা ଯିବ। Pending ଥିଲେ MIS ଙ୍କ approval ଅପେକ୍ଷା କରନ୍ତୁ।
🚫 Auto-Rejection Rules
ନିମ୍ନ ନିୟମ ଭଙ୍ଗ ହେଲେ ଛୁଟି ତୁରନ୍ତ Auto-Reject ହେବ:
① Casual Leave — ଛୁଟି ଦିନ ର 1 ଦିନ ଆଗରୁ apply କରିବାକୁ ହେବ। ଆଜି / past date ରେ apply କଲେ ABSENT mark ହେବ।
② Sick Leave — future date ରେ apply ହୁଏ ନାହିଁ, କେବଳ ଆଜି ବା past ରେ apply କରନ୍ତୁ।
③ Emergency Leave — future date ରେ apply ହୁଏ ନାହିଁ, ଏହା plan ହୁଏ ନାହିଁ।
④ SPL — MIS + Store Manager ଉଭୟଙ୍କ approval ଦରକାର।
